# Week3 0812 HW01
## API docs (詳細傳送資料可見 Postman json 檔案)
### GET 拿到所有 order 的資料

- http://localhost:8080/api/v1/order

### GET 利用 id 拿到 order 的資料

- http://localhost:8080/api/v1/order/:id
    - id: orderID 

### POST 傳送 json 來新增 order

- http://localhost:8080/api/v1/order

### PUT 利用 id 來更新 order

- http://localhost:8080/api/v1/order/:id
    - id: orderID 

### DELETE 利用 id 來刪除 order

- http://localhost:8080/api/v1/order/:id
    - id: orderID
