# API (詳細傳送資料可見 postman json 檔案)
## GET 查詢賣家賣場的商品
- http://localhost:8080/store/:name
    - name: 賣家名字 


## POST 新增賣家的商品
- http://localhost:8080/store/:name
    - name: 賣家名字


## PUT 更新賣家的商品資料
- http://localhost:8080/store/:name/:id
    - name: 賣家名字 
    - id: 商品id


## DELETE 刪除賣家的商品
- http://localhost:8080/store/:name/:id
    - name: 賣家名字 
    - id: 商品id
